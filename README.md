# Project Setup Guide

Follow the steps below to run the project locally.

## Prerequisites

- Node.js (v18+ recommended)
- npm

## Setup Steps

1. Take a pull from the `master` branch.
2. In the **root folder**, rename `.env.example` to `.env`.
3. In the **frontend folder**, rename `.env.example` to `.env`.

## Installation & Build

From the **root folder**, run:

```bash
npm run install:backend
npm run install:frontend
npm run build
npm start
```
